A Retrieval-Augmented Generation (RAG) app built with Python, Streamlit, and ChromaDB to help users query technical documentation.

## Features
- Semantic search using Sentence Transformers
- Streamlit UI for interaction
- ChromaDB vector storage for embeddings

## How to run
1. Clone the repo
2. Create a virtual environment
3. Install requirements: `pip install -r requirements.txt`
4. Start: `streamlit run day2_app.py --server.headless true`
